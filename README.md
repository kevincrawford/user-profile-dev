# SPEDxchange

> The Community for Special Education Answers

## Quick Start

```
# Configure Environments
```

```bash
# Install server dependencies
npm install

# Install client dependencies
cd client
npm install

# Run both Express & React from root
npm run dev

# Build for production
npm run build
```
